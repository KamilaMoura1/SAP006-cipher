A cifra de César é uma das técnicas mais antigas de cifrar uma mensagem. É um tipo de cifra por substituição, em que cada letra do texto original é substituida por outra que se encontra há um número fixo de posições (deslocamento) mais a frente do mesmo alfabeto.

Por exemplo

Quero cifrar a palavra LABORATORIA com um deslocamento de 5, logo, o código para a palavra será: PEFSVEXSVLE

Com o "Le code Secret" conseguirá criar mensagens secretas ou decifra-las, baseado em um dos primeiros tipos de criptografia da história, um verdadeiro Sherlock Holmes!
